## Introduce
聚合，提供了像是RDBMS的COUNT、AVG等數學運算的方法，也提供了像是RDBMS裡的Group By等方法

## Bucket Aggregation
滿足特定條件的document集合，像是RDBMS裡的Group By

## Metric Aggregation
提供數學運算，min / max / sum / avg / cardinality 等

##　Pipiline Aggregation
對其他Aggregation再進行一次Aggregation
